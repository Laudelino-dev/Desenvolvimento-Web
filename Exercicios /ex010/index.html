Usando links externos

Para utilização de links utilizamos o par de tags a, e com ele o atributo href, dentro desse atributo devemos colocar o link do respectivo site.
Ex: Esse é o meu repositorio no Github
Para âncora também temos outro atributo, que define se a pagina será aberta na mesma aba ou em uma aba diferente. Para links externos sempre devemos abrir em uma aba diferente, Isso será definido no atributo target, onde possui dois tipos de parametros:

  • _blank - Para abrir em uma nova aba
  • _self - Para abrir na mesma aba

Também possui o atributo rel, que indica qual a natureza do link :
  • next - Indica que o link é para a próxima parte do documento atual
  • Prev - Indica que o link é para a parte anterior do documento atual
  • author - Indica que é um link para o site do autor do artigo
  • external - Indica que é um link para outro site que não faz parte do site

Links internos

Esse conceito é utilizado para fazer ancoras entre pasta internas como ja diz o proprio nome.
Para fazer essa ligação, na parte em href não é necessario colocar o link completo, basta colocar o caminho da pagina dentro da sua maquina.
Ex: Está aqui é a minha segunda pagina

Links para downloads

Para links para downloads devemos adicionar 2 atributos, o atributo Download onde nele iremos colocar o nome do arquivo que será baixado, e o atributo type onde nele iremos colocar o tipo de arquivo que será baixado.

  • Baixar meu livro em PDF
  • Baixar meu livro em ZIP
