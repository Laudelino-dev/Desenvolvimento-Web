Listas ordenadas

Listas ordenadas são listas onde a ordem da sequencia vai importar, por exemplo: numa receita de bolo ou algo parecido.
Para fazer uma lista ordenada utilizamos o par de tags <ol>, e dentro delas os itens seram listados em um par de tags <li>
Onde vai haver 2 parametros, o parametro type onde vai ser definido o tipo de contagem da lista,
Existem 4 tipos de "type":

  1. 1 - Onde a contagem vai ser os numeros
  2. A - Onde a contagem vai ser de Letras em maiusculas
  3. a - Onde a contagem vai ser de letras em minusculas
  4. I - Onde a contagem vai ser em romanos em Maiusculas
  5. i - Onde a contagem vai ser em romanos em minusculas
E o start onde vai ser definido de onde a contagem vai iniciar

Lista não ordenadas

Lista não ordenadas são totalmente o contrario das ordenadas, aqui não importa a ordem, so basta querer uma lista tbm.
Para criar basta utilizar o par de tags <ul>, e da mesma forma que e a anterior, os itens tem que ir dentro do par de tags <li>
Ela possui somente o parametro type que tem 3 definições.
Ex:

  • disk - padrão, uma bolinha preta
  • circle - Uma bolinha com uma borda preta e sem preenchimento
  • square- um quadrado totalmente preto

Lista de definição

Uma lista de definição é como se fosse um dicionario, Você define a lista usando o par de tags <dl>, que significa Definition list
Adiciona o possivel termo com o par de tags <dt>, que significa Definition term
E apos adiciona a descrição do possivel termo em um par de tags <dd>, que significa Definition description
ex:

HTML
Linguagem de marcação de texto voltada para criação de paginas
CSS
Linguagem de marcação que serve para criar design
JavaScript
Linguagrm de programação para interação web
