Citações e códigos

código-fonte/Pré-formatação

Utilizado para mostrar alguma linha de codigo na pagina web, Utilizamos essa forma pois deixa o codigo em forma mono-espaçado e padrão! Para isso usamos o par de tags <code>

Exemplo: num = int(input('Digite um numero')) if num % 2 == 0: print(f' o número {num}é PAR'); else: print(f' o número {num} é IMPAR'); print('Fim do programa');

Como podemos ver acima ele fica de uma forma padrão e mono-espaçado,mas completo em linha. Existe outro par de tags que podemos deixar essa linha de codigo, como Pré-formatado da forma escrita na sua IDE. Para isso utilizamos o par de tags <pre> 

Exemplo:
        num = int(input('Digite um numero'))
    if num % 2 == 0:
        print(f' o número {num}é PAR');
    else:
        print(f' o número {num} é IMPAR');
    print('Fim do programa');

Citações simples

Para citações simples usamos o par de tags <q>
Pois ele serve para uma breve e rapida citação de algum autor ou frase de terceiros que seja utilizada em linha.

Ex: Eu tenho um amigo que diz que: É melhor um pássaro na mão do que dois voando

Citações composta

Para citações compostas usamos o par de tags <blockquote>
Pois ao contrario da simples, ela usada para uma mais completa, deixando ela com um padrão unico e mais recuado.

Ex:

a excelência não é um ato isolado ou um golpe de sorte, mas sim o resultado da repetição consistente de boas ações.

Abreviações

Utilizamos o par de tags <abbr>.
Onde assim que a utilizarmos na parte de "tittle" iremos colocar o nome completo da palavra abreviada que adicionamos e com isso, assim que passarmos o cursor por cima dessa abreviação, irá mostar o nome completo da palavra.

Ex: Eu estou estudando HTML e CSS.

Listas

Possuimos 3 tipos de listas

  • Listas ordenadas
  • Listas não ordenadas
  • Listas de definições

Listas ordenadas

Listas ordenadas são aquelas onde a sequencia da lista importa! Ex: Receitas de bolo ou algo parecido.
Para isso devemos utilizar o par de tag <ol>, e dentro delas para adicionarmos os itens da lista devemos adicionar no par de tags <li>.
Para listas ordenadas temos 5 tipos de identificação numerica.

  1. 1 - Para contagem Numerica - PADRÃO
  2. A - Para contagem em letras maisculas
  3. a - Para contagem em letras minusculas
  4. I - Para contagem em romanos em maisculas
  5. i - Para contagem em romanos em minusculas

Listas não ordenadas

Listas não ordenadas são aquelass onde a sequencia não vai importar tanto.
Para isso devemos usar o par de tags <ul>, e da mesma forma que as ordenadas devemos adicionar os itens dentro do par de tags <li>.
Para listas ordenadas temos 3 tipos de identificação.

  • disk - Para adicionar uma bolinha preta - PADRÃO
  • circle - Para adicionarmos um bolinha com meio vazio
  • square - Para adicionarmos um quadrado preto

Lista de definições

Funciona como se fosse um dicionario, onde se tem um termo e uma descrição para o proprio termo.
Para utilizarmos devemos adicionar primeiro o par de tags <dl>, para a criação de uma lista de definição.
Dentro dela devemos adicionarmos o par de tags <dt>, para definirmos o termo que terá sua descrição.
A seguir devemos adicionarmos a descrição com o par de tags <dd>, para definirmos a descrição do termo em si.
